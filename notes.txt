#styles props

we can pass the styles in props so that we have to do 

define the type like..

type StyleProps={
    "styles":React.CSSProperties
}

pass the props to div or element in style data tag

<ComponentName style ={{"styleName":"vale"}}

<span style={props:StyleProps}> hai this is abs </span>
